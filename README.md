> 幼小衔接冲刺版 · 每日打卡 · 积分兑好东西

## 在线访问

**GitHub Pages 地址（上线后填写）：**  
`https://你的用户名.github.io/fxc-checkin/`

## 功能说明

| 功能 | 说明 |
|------|------|
| 累计积分看板 | 显示总积分、本周、本月、打卡天数 |
| 近7天连续打卡 | 圆点可视化展示 |
| 每日打卡提交 | 15个项目，手机端点击勾选，一键提交 |
| 历史记录 | 查看最近30天打卡情况 |
| 积分超市 | 17个兑换项目，灰色=积分不足 |

**数据存储：** 全部存在手机/浏览器本地（localStorage），零后端，免费。

## 上线步骤（5分钟搞定）

### 第一步：注册 GitHub

访问 https://github.com 注册账号（免费）

### 第二步：新建仓库

1. 点击右上角 **+** → **New repository**
2. 仓库名填：`fxc-checkin`
3. 勾选 **Public**（必须公开才能免费用 Pages）
4. 点击 **Create repository**

### 第三步：上传文件

1. 进入仓库页面
2. 点击 **uploading an existing file**
3. 把这个文件夹里的 `index.html` 拖进去
4. 点击 **Commit changes**

### 第四步：开启 GitHub Pages

1. 进入仓库 **Settings** → 左侧 **Pages**
2. Source 选 **Deploy from a branch**
3. Branch 选 **main**，目录选 **/ (root)**
4. 点击 **Save**

### 第五步：访问

等1-2分钟，访问：  
`https://你的GitHub用户名.github.io/fxc-checkin/`

把网址保存到手机桌面，像APP一样使用！

- 建议每周截图备份一次积分情况
