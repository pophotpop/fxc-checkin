# 范鑫澄成长积分 🌟

幼小衔接冲刺版 — 每天努力，积分兑好东西。

## 功能

- 🌟 **积分看板**：累计/本周/本月积分 + 近7天打卡连续记录
- ✅ **今日打卡**：15个项目，分4组，手机点选提交
- 🛒 **积分超市**：17个兑换项（屏幕时间/玩具/体验活动/零食）
- 🔐 **家长锁**：管理员可打卡+兑换，访客只能查看
- 📊 **历史记录**：最近30天，打卡得分+兑换记录

## 默认管理员密码

`202106`（范鑫澄生日年月，上线后可自行修改 `ADMIN_PIN` 变量）

## GitHub Pages 上线

1. 注册 [GitHub](https://github.com)
2. 新建仓库 → 名字 `fxc-checkin`，勾选 Public
3. 上传所有文件（index.html、avatar.png、README.md）
4. 仓库 **Settings → Pages** → Source 选 `main` → Save
5. 约1分钟后访问 `https://你的用户名.github.io/fxc-checkin/`

## 管理 vs 访客

| 操作 | 管理员 | 访客 |
|------|--------|------|
| 看积分/打卡状态 | ✅ | ✅ |
| 看积分超市 | ✅ | ✅ |
| 看历史记录 | ✅ | ✅ |
| 打卡勾选+提交 | ✅ | ❌ |
| 兑换积分 | ✅ | ❌ |
| 清除数据 | ✅ | ❌ |

右下角点击 🔒 图标输入密码即可切换管理模式。

## 技术

纯静态 HTML + localStorage，零后端、零数据库、零费用。
